# Shared_Data

This folder holds data files used by the AFP Estimator app.

Expected files:
- companies.csv
- locations.csv
- FY2026_GSA_ZipCodeFile.csv
- 2026-01_Dept-of-State_PerDiem_PD.csv
- Templates/*.md
- afp_logo.jpg (optional)
- iso_logo.jpg or iso_logo.png (optional)

If these files are not present, the app will run with limited data.
